# Ghost Grid 👻

A 5-level arcade maze escape game built with Python and Pygame.

## About
Navigate through hand-crafted mazes while two ghosts hunt you down. 
One chases. One patrols. Neither forgives mistakes. Reach the golden 
exit on each level to advance. Beat all five floors to win.

## Gameplay
- 5 unique maze layouts with increasing complexity
- Two ghost types: Chaser (tracks your position) and Patrol (roams randomly)
- Pixel art player and ghost sprites drawn entirely with Pygame primitives
- Win/lose/level-complete game states
- Press R to retry or advance levels

## Controls
| Key | Action |
|-----|--------|
| W / ↑ | Move Up |
| S / ↓ | Move Down |
| A / ← | Move Left |
| D / → | Move Right |
| R | Retry / Next Level |

## Built With
- Python 3
- Pygame

## Installation
```bash
pip install pygame

```
